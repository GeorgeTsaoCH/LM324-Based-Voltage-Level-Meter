# LM324-Based Voltage Level Meter

Analog voltage level indicator designed and tested at Seneca Polytechnic.  
This project uses an **LM324** as a comparator, a **resistor-divider threshold network**, and **4 LEDs** to indicate input voltage level ranges.

## Highlights
- Designed and tested an **LM324-based voltage level indicator**
- Used **OrCAD/PSpice** to simulate threshold behavior before hardware build
- Performed **breadboard validation** and **PCB implementation**
- Completed **root cause analysis** to identify the LM324 input range limitation
- Used **DMM continuity testing** and systematic troubleshooting to verify final PCB operation

## Project Overview
The circuit compares an input voltage against multiple reference voltages generated by a resistor divider.  
Each comparator stage drives an LED when the input exceeds the corresponding threshold.

This project was valuable not only as a design exercise, but also as a **hardware debugging and validation** exercise:
- comparing **nominal vs. adjusted threshold values**
- checking expected vs. measured behavior
- reviewing the **LM324 datasheet**
- correcting the design after discovering practical device limitations

## Key Engineering Lessons
1. **Simulation is useful, but real component limits matter**  
   The original design needed adjustment because the LM324 input range does not extend close to the supply rail.

2. **Debugging should be systematic**  
   Possible causes such as poor wiring, faulty breadboard contact, and damaged components were checked before identifying the actual root cause.

3. **Validation is part of design**  
   Final resistor values were adjusted to bring the highest threshold into a usable operating range.
